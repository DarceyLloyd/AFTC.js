Usage

You can set its configuration each time you use it and use it like the old way eg:

  • logTo("Hello World 1",{elementId:"output",clear:true,append:true,logToConsoleAlso:true,addLineBreaks:true});

  • logTo("Hello World 2",{element:myHTMLElement,clear:true,append:true,logToConsoleAlso:true,addLineBreaks:true});

  • logTo("Hello World 3",{elementId:"output"});

  • logTo("Hello World 4",{element:myHTMLElement});

or you can set it's configuration before use and forget about setting it's options till you have to, eg:

  • logTo({elementId:"output",clear:true,append:true,logToConsoleAlso:true,addLineBreaks:true});

  • logTo("Hello World 1";

  • logTo("Hello World 2";

  • logTo("Hello World 3";

Persistive options / options which will remember their settings are:

  • element : HTML Element Object
  • append:boolean (default: true)
  • addLineBreaks:boolean (default: true)
  • logToConsoleAlso:boolean (default: true)
output1 - this will be used for logTo (you wont see this if clear is set to true)
output2 - this will be used for logTo (you wont see this if clear is set to true)
output3 - this will be used for logTo (you wont see this if clear is set to true)
